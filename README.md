# UTM Templates

### A few free preinstalled templates for use with UTM on Mac with Apple Silicon.

## Usage

Download the app from releases and drag the app in the DMG to your applications folder.

## Current VMs Provided
* Windows 10
* Windows 11
* macOS 12
* Ubuntu Linux
* Fedora Linux
* OpenSUSE Linux
